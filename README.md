## Environment
- 언어: Go(v1.17)
- 프레임워크: echo(v4)
- ORM: entgo(v0.9.1)
- 데이터베이스: Postgresql(v13.2)
- 개발 환경: Docker(v20.10.8) & Docker-Compose(컨테이너들 정의 및 관리)
